Fixing Broadcom wifi driver issues in ubuntu, debian kernels.

Techie



FIX: Broadcom wifi driver issues.

Today I am going to show you guys how to fix the wifi driver issue which most of the new laptops today which installs any new distro of ubuntu or Debian. Here I am going to show how to fix the error faced when the network card is of Broadcom. This error is faced by every laptop which has new wifi-card with Linux installed irrespective of wifi card. Ie, even laptops with Realtek wifi-driver face this issue . But take care, fixing it is done in a different manner. If needed I will post that in future.

Before getting started, let me tell you guys that fixing the error in Broadcom driver is really easy compared to Realtek and intel wifi cards.

So letâ€™s start.

#1 : First of all, get your system up-to-date.

$ sudo apt-get update

#2: Next, what we are going to do is that we are going remove the installed driver at first and then move on.

$ sudo apt-get purge bcmwl-kernel-source



#3: So as we have removed the driver now we are going to re-install the driver.

$ sudo apt-get install linux-generic bcmwl-kernel-source


#4: Done. reboot your system now.

$ sudo reboot

Hope this will work for you guys. If not comment below.

Cheers.


Note: If you don't know which wifi card you have, then do this in the terminal(Ctrl+Alt+T)

$ lspci

This will give your network controller name.
